Pre-push hook runs `tests/replay/check.sh` (tier-2). `.pbstream` goldens are gitignored; regen on legitimate behavior change with `cp *.check.pbstream *.pbstream`.
Local builds are Qt6 (`build/`); Qt5 is gated by CI (Ubuntu + Fedora × Qt5/Qt6 matrix in `.github/workflows/ci.yml`). To smoke-test Qt5 locally: `rm -rf build && cmake -B build -DSEQ_USE_QT5=ON`.
Qt5/Qt6 string-view portability: `QXmlStreamReader::name()` and `QXmlStreamAttribute::value()` return `QStringRef` on Qt5 and `QStringView` on Qt6 (the type was renamed, not aliased). Use `auto` for the local, or call `.toString()` immediately, to compile against both.
Qt5 `QColor(QString)` parses SVG/CSS3 named colors (e.g. `gray=#808080`, `green=#008000`, `darkBlue=#00008b`) — NOT the `Qt::GlobalColor` enum (`Qt::gray=#a0a0a4`). `src/seqcolor.cpp`'s 148-entry table is the canonical list; `mapcolors.h` feeds 64 of those names through `pSEQPrefs->getPrefString(...)` into colors.
Custom QVariant types via `Q_DECLARE_METATYPE(T)` (e.g. `SeqColor`): use `if (v.userType() == qMetaTypeId<T>())` for read dispatch (can't go in a `case` — runtime id), and `QVariant::fromValue<T>(v)` for construction (the `QVariant(T)` ctor rejects user metatypes).
If `AutoMoc warning: includes the moc file "X.moc"...` reappears, it's the qmake `#ifndef QMAKEBUILD / #include "X.moc" / #endif` block sneaking back in. CMake AutoMoc generates `moc_X.cpp` from the header's `Q_OBJECT`; the cpp's `.moc` include is needed only for inline-in-cpp `Q_OBJECT` (which the daemon doesn't use). Just delete the block.
EQMac wire framing is per-(srcPort, dstPort) tuple, not per-EQPacketStream: the host-wide BPF filter funnels multiple flows (active zone server, next zone server during transition, stray non-EQ UDP) into one stream. `EQPacketStream::m_decoders` keys `EQMacDecoder` by tuple — sharing one decoder corrupts ARQ-dedup state across flows.
EQOldStream retransmits the same dwARQ with fresh dwSEQ until ack'd; `EQMacDecoder::m_lastArq` dedups on signed-int16 delta (handles 0xFFFF→0 wrap). Don't remove this thinking it's redundant with the post-EQMac ARQ cache in `packetstream.cpp` — that cache only runs in `processPacket()`, which the EQ Mac path bypasses.
`cmake --build build --target setcap` restores cap_net_admin,cap_net_raw via passwordless sudoers rule. Re-run after every relink before live capture.
